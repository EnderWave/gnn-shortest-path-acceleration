## 汇报维护规则

1. 每完成一个阶段，先补齐对应汇报，再修改本页状态。
2. 结果数字必须链接到 `results/` 中可追溯的 CSV 或报告。
3. “脚本完成”和“实验完成”分开记录；没有结果文件时不能标记为实验完成。
4. 新发现的限制和负结果也要记录，不能只保留正向结论。
