# Porto OD + OSM 数据说明

这个目录用于记录 Porto 真实 OD 查询负载实验的数据来源。原始大文件目前保存在
`data/compressed/porto/`，后续处理出的 OD 样本、路网图和节点映射可以放到
`data/processed/porto/`。

## 已下载文件

| 文件 | 说明 | 大小 |
| --- | --- | ---: |
| `data/compressed/porto/uci_porto_taxi.zip` | UCI Porto taxi 数据集完整下载包 | 510 MB |
| `data/compressed/porto/train.csv.zip` | 主训练集压缩包，内部为 `train.csv` | 509 MB |
| `data/compressed/porto/portugal-latest.osm.pbf` | Geofabrik Portugal OSM PBF，包含 Porto 路网 | 398 MB |

## 数据来源

- Porto taxi trajectory dataset: UCI Machine Learning Repository, Taxi Service Trajectory - Prediction Challenge, ECML PKDD 2015.
- OSM road network source: Geofabrik Portugal extract, based on OpenStreetMap data.

## 后续处理建议

1. 不要直接解压 `train.csv.zip` 到仓库里，解压后约 1.9 GB。后续脚本可以从 zip 里流式读取。
2. 从每条 taxi 轨迹的 `POLYLINE` 中取第一个 GPS 点作为 origin，最后一个 GPS 点作为 destination。
3. 从 `portugal-latest.osm.pbf` 裁剪或抽取 Porto driving road network。
4. 把 origin/destination GPS 点吸附到最近的路网节点，生成 OD 查询表。
5. 先抽样 10,000 条 trip 检查吸附成功率、OD 分布和热点是否明显。

## 当前已生成结果

| 文件 | 说明 |
| --- | --- |
| `data/processed/porto/波尔图起终点样本_10万.csv` | 从 `train.csv.zip` 流式抽取的 10 万条有效 OD 样本 |
| `data/plots/波尔图起终点热力图_10万.png` | 不带道路底图的起点/终点热力图 |
| `data/plots/波尔图道路底图起终点热力图_10万.png` | 使用本地 OSM PBF 抽取道路底图后的起点/终点热力图 |
| `data/processed/porto/波尔图道路节点.csv` | 从 OSM PBF 抽取的 Porto 范围道路节点 |
| `data/processed/porto/波尔图道路边.csv` | 从 OSM PBF 抽取的 Porto 范围有向道路边，边权为米 |
| `data/processed/porto/波尔图起终点节点查询_10万.csv` | 10 万条 OD 经纬度吸附到最近道路节点后的完整结果 |
| `data/processed/porto/波尔图可用起终点节点查询_200米.csv` | 起点和终点吸附距离都不超过 200 米的可用查询 |
| `data/processed/porto/波尔图起终点吸附质量报告.md` | 路网规模、吸附成功率和吸附距离统计 |

对应脚本：

| 脚本 | 作用 |
| --- | --- |
| `scripts/生成波尔图起终点热力图.py` | 从 Porto taxi 压缩包中抽取 OD 并绘制普通热力图 |
| `scripts/生成波尔图道路底图起终点热力图.py` | 从本地 OSM PBF 抽取道路底图，并叠加 OD 热力图 |
| `scripts/抽取波尔图路网并吸附起终点.py` | 抽取 Porto 路网节点/边，并把 OD 经纬度吸附到最近道路节点 |

运行带道路底图的脚本需要使用项目虚拟环境：

```bash
.venv-linux/bin/python scripts/生成波尔图道路底图起终点热力图.py
```

抽取路网并吸附 OD：

```bash
.venv-linux/bin/python scripts/抽取波尔图路网并吸附起终点.py
```

当前吸附结果概要：

- 路网节点数：133,839
- 有向边数：221,589
- 原始查询数：100,000
- 200 米内可用查询数：98,082
- 200 米内可用占比：98.08%
